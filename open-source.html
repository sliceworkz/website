Sliceworkz Eventstore Library

An open source eventstore implementation in Java. Lightweight, minimal dependencies, with PostgreSQL-backed storage for production and in-memory storage for development and testing.

✓

Fully compliant with DCB (Dynamic Consistency Boundary)

✓

Fully typed access to events via EventStreams

✓

Event query capabilities on event types and tags

✓

Optimistic locking on event appends

✓

Built-in upcasting of legacy events
